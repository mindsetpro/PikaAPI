PikaAPI Documentation

Welcome to the PikaAPI documentation. Here you will find information on how to use our API to access various Pokémon data.

To get started using the PikaAPI, you will need to sign up for an API key. Once you have your key, you can start making requests to our endpoints.

Our API provides several endpoints that allow you to access information about Pokémon generations, types, berries, and more. Refer to the specific endpoint documentation for details on how to use each endpoint.

For most endpoints, you will need to include your API key in the request headers for authentication. Make sure to include the key in the 'Authorization' header with the value 'Bearer YOUR_API_KEY'.
