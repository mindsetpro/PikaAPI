PikaAPI Documentation

Welcome to the PikaAPI documentation. Here you will find information on how to use our API to access various Pokémon data.

No API key is required to access PikaAPI.

Our API provides several endpoints that allow you to access information about Pokémon generations, types, berries, and more. Refer to the specific endpoint documentation for details on how to use each endpoint.

Below are some examples demonstrating how to make requests to our API using various programming languages:

JavaScript Example:

fetch('https://pikaapi.com/data/gens.json')
    .then(response => response.json())
    .then(data => console.log(data));

Python Example:

import requests

response = requests.get('https://pikaapi.com/data/types.json')
data = response.json()
print(data)

Ruby Example:

require 'net/http'
require 'json'

url = URI('https://pikaapi.com/data/pokemon.json')
response = Net::HTTP.get(url)
data = JSON.parse(response)
puts data
